## Technologies
- NodeJS
- Node Package Manager (NPM)
- Hapi module
- Nanoid module
- Nodemon module

## Install
- Install NodeJS.
```
sudo apt install nodejs
```
- Install Node Package Manager (NPM).
```
sudo apt install npm
```
- Install NPM modules.
```
npm install
```

## Run
```
npm run start
```
